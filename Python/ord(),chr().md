#### 📌 아스키(ASCII) 코드
+ 미국 정보 교환 표준 부호 (American Standard Code for Information Interchange)
+ ```기호```와 ```영문알파벳```에 적합한 문자 인코딩
+  ```7비트``` 인코딩
    - 0 ~ 127

## ord()
```문자열```을 ```아스키코드```로 변환하는 함수
```python
# 사용 예
ord('a') # == 97
ord('b') # == 98
```

## chr()
```아스키코드```를 ```문자열```로 변환하는 함수
```python
# 사용 예
chr(97) # == 'a'
chr(122) # == 'z'
```
